# 504. Base 7

### Easy

Given an integer num, return a string of its base 7 representation.

### Example 1:

Input: num = 100
Output: "202"

### Example 2:

Input: num = -7
Output: "-10"

Constraints:

-10^7 <= num <= 10^7

```go
// O(n) time，先都按照正数处理。负数加负号即可。
func convertToBase7(num int) string {
   if num == 0 {
      return "0"
   }
   n := num
   if n < 0 {
      n = -n
   }
   res := []byte{}

   for n > 0 {
      res = append([]byte{byte(n%7) + '0'}, res...)
      n = n / 7
   }
   if num < 0 {
      res = append([]byte{'-'}, res...)
   }
   return string(res)
}
```
